Основы синтаксиса

урок №1
  • Точка с запятой
    Не обязательна, если каждое свойство прописывается с новой строки. Рекомендуется всегда ее ставить.
  • Переменные
    Существует несколько типов. Фиксированные значения - литералы, значения констант и значения переменных.
  • Операторы
    Существует большое множество - они означают операцию, которую нужно выполнить.
    Например * / + - = и т.д.
  • Выражения
    Это комбинации значений, переменных и операторов, которые либо присваивают переменной значение, либо возвращают какое-то значение без его присваивания.
  • Ключевые слова
    Они предназначены для определения действия, которое необходимо выполнить. Например let num = 4 говорит браузеру создать переменную
  • Блоки инструкции
    Сгруппировывает инструкции с помощью function name() {}
  • "use strict";
    По умолчанию новые функции JS отключены и для их работы необходимо прописывать данную команду в самом начале кода

Переменные

урок №2
  • Обьявление переменной
    Для этого используется ключевое слово let
  • Наименование
    Для нейминга используют lowerCamelCase - каждое слово начинается с большой буквы кроме первой
  • Присвоение значений
    После обьявления переменной нет необходимости сразу задавать ей значение. Это можно сделать позже.
    Обьявить несколько переменных можно используя только один let и задать их через запятую.
  • "use strict";
    Без данной строки переменные можно обьявлять без ключевого слова let.
  • Область видимости переменных
    Если обьявить переменную в пределах одного блока за его пределами она не существует.
  • Почему не
    var
    ?
    Отличие в том, что ее можно использовать до обьявления ее значения.
    Область видимости выходит за пределы всех блоков кроме функций, например if

Типы данных

урок №3

JS является динамически типизированным языком. То есть тип данных меняется динамически в момент присвоения значения.

  • Определение типа
    С помощью оператора
    typeof
    можно определить тип переменной.
  • undefined

    Неопределенный тип - присваивается переменным, которым не задали значение.
  • null

    Ноль - значение переменной задается в случае, если мы обратились к обьекту которого нет.
  • boolean

    Правда/ложь.
  • number

    Числовой тип данных - представляет собой целые и дробные числа.
    infinity
    - бесконечность является типов number образуется при делении на ноль. Может быть положительным и отрицательным.
    Nan
    обозначает вычислительную ошибку.
  • BigInt

    Очень большое число. Для его получения к числовому значению необходимо добавить букву
    n
    .
  • String

    Строка может быть обернута одинарными двойными и обратными кавычками.
    Между двойными и одинарными разницы нет, а с помощью обратных можно встраивать специальные выражения.
  • Object

    Сложные типы данных, их можно создать используя {} со встроенными значениями ключа и его значения.
  • Symbol

    Уникальный идентификатор обьекта. Создается с помощью одноименной функции.
    let id = Symbol('id');
  • function

    Тот же самый обьект, но для более простого определения функции.
  • Преобразование типов
    String(var)
    - перевод из числа в строку.
    Number(var)
    - перевод из строки в число.
    Boolean(var)
    - перевод в boolean.
    Есть так же автоматическое преобразование типов, например при делении "2" / "2".

Операторы
JavaScript

урок №4
  • Типы операций
    Существуют 2 типа операций:
    бинарный
    - применяется к двум операндам,
    унарные
    - к одному операнду.
  • Специальные возможности
    +
    - применяется и к строкам
    Если в выражении есть строка, то результат ее сложения будет строкой.
    Операции до сложения со строкой значения не имеют.
    Все другие операторы работают только с числами и превращают строки в числа. br Унарный оператор сложения выполняет роль преобразования строки в число.
    Number()
    - явное преобразование в число
  • Инкремент/Декремент
    Инкремент
    ++
    - увеличивает на единицу.
    Декремент
    --
    - уменьшает.
    Работает данная функция только с переменной, располагать их можно, как до, так и после переменной.
  • Оператор "
    ,
    " используется редко, и позволяет вести запись (1 + 3, 2 + 4), результат будет последнее выражение, равное 6.
  • Операторы сравнения
    больше, меньше, == равно, !== не равно, === строгое равно, !== строгое не равно.
    Если сравнивать строки, то
    JS
    сравнивает по индексам их букв, Б больше А.
    Регистр маленькой буквы считается больше чем большой, а больше А.
    При сравнении строк и чисел
    JS
    будет приводить их к числовым значениям. Логические значения true становятся 1, а false - 0.
    Для проверки значений без приведения типов выполняется через строгое равно/ не равно.
  • Логические операторы
    || - логическое или
    && - логическое и
    Приоритет оператора и больше, чем у или.
    ! - не. Приводит аргумент к логическому типу и возвращает противоположное значение. ?? - оператор обьдинения с null, возвращает первое значение, если оно не равно 0

For, while. Break, continue

урок №6
  • while (условие) {тело цикла}
    Проверяет условие перед, каждой итерацией. Для создания бесконечного цикла используют while (true)
  • do while
    Используется, если необходимо хотя бы одно выполнение, не зависимо от условия. Проверяет условия после каждой итерации.
  • for (... ; ...; ...) {}
    Любую часть цикла мы можем пропустить и вписать ее внутри {}.
    let i = 0;
    for (; i == 1; i++) {}
  • break
    for (... ; ...; ...) {if (nun == 2) break;}
    Используется для прерывания циклов
  • continue
    for (... ; ...; ...) {if (nun == 2) continue;}
    при выполнении данного условия выполнения круга прерывается и переходит дальше.
    Если на текущей итерации цикла нечего делать не нужно, то переходим к следующей.
  • Метки
    имя метки: for (... ; ...; ...) {for (... ; ...; ...) {break имя метки}}
    позволяет управлять break, continue на разных уровнях вложенности циклов.
